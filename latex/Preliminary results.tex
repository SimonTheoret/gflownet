\begin{abstract}
Le but de notre projet est d'entraîner un modèle génératif dans le but de simuler des parties d'échec. Le modèle génératif sera basé sur le GFlowNet. Notre modèle utiliseras des librairies python déjà développées, telle que \textit{python-chess}, dans le but de simuler les actions possibles durant un match d'échec. De plus, notre modèle GFlowNet utiliseras comme fonction de récompense le moteur libre d'échec \textit{Stockfish}.

Nous avons les objectifs suivants, en ordre décroissant d'importance:
\begin{enumerate}
    \item Entraîner avec succès un modèle GFlowNet
    \item Créer une interface graphique pour visualiser les actions du modèle
    \item Rendre l'interface graphique interactive
    \item Évaluer le pointage \textit{elo} (\textit{elo score}) de notre modèle
\end{enumerate}

\end{abstract}
